Connect Orbeon and Odoo

The Orbeon Server under control in Odoo

In Odoo, setup and manage the "so-called" Orbeon Persistence Server.
This handles the bi-directional connection and storage between Orbeon and Odoo.

"For the tech-savvy"
The Orbeon Persistence Server is a WSGI/Web server, which is included and managed by this App.

List

List Orbeon Server connection records.

Create and Update

Manage settings of the Orbeon Server interface in Odoo.

Start the connection

Start the Orbeon Persistence Server connection in Odoo.

Stop the connection

Stop the Orbeon Persistence Server connection in Odoo.
